# React UB

### To run


```
> $ git clone https://github.com/alicoding/react-webpack-babel.git
```

Then install the dependencies:

```
> $ npm install
```

start your app :

```
> $ npm start
```

You can simply run build : 

```
> $ npm build
```

If you want to run test run this command: 

```
> $ npm run test
```
